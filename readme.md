---

## 🧠 Project Description

The **Anonymous University Problem Portal** is designed to help students submit issues they face — academic or campus‑related — without fear of judgment or exposure.

Students can:

- Share difficulties with subjects or exam preparation  
- Report issues with administration or documentation  
- Express concerns about campus facilities  
- Discuss hostel/accommodation problems  
- Raise awareness about scams or harassment

All problems are posted anonymously to protect privacy.

---

## ✨ Features

- 🎭 **100% Anonymous** — No personal name or roll number is collected  
- 📝 **Problem Submission Form** — Students can submit problems easily  
- 📊 **Problem Display** — View problems submitted by others  
- 🗂 **Category Filtering** — Filter problems by type  
- 📅 **Date & Time Stamp** — Each problem records when it was posted  
- 📦 **LocalStorage Support** — Uses browser storage to save the data locally
